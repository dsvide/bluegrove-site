# Blue Grove — Website

## Setup

1. Install Node.js from https://nodejs.org (LTS version)
2. Open terminal in this folder
3. Run: npm install
4. Run: npm run dev
5. Open http://localhost:5173

## Deploy to Vercel

1. Create account at https://vercel.com
2. Push this folder to GitHub
3. Connect repo to Vercel
4. Done — site is live!

## Connect Shopify Buy Button

1. In Shopify admin → Settings → Apps → Buy Button
2. Create a Buy Button for your product
3. Copy the JavaScript code
4. Paste it in src/App.jsx in the Shop component

## Add your real photos

Replace the URLs in the IMG object at the top of src/App.jsx
with your own image URLs or local file paths.
